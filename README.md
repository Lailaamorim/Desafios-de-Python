## 🧠 Sobre o Python 3

Python é uma linguagem de programação criada por Guido van Rossum.

Ela é conhecida por:

- Sintaxe simples e organizada
- Fácil leitura
- Grande comunidade
- Versatilidade
- Linguagem de alto nível
- Multiplataforma (Windows, Linux e macOS)
